关于 TypeScript

TypeScript 是 JavaScript 的一个超集，主要提供了类型系统和对 ES6 的支持，它由 Microsoft 开发，代码开源在github上。 http://www.typescriptlang.org/

TypeScript 增加了代码的可读性和可维护性

可以让编译阶段就可以发现大部分的错误，让你运行的时候不同意出错，而且看类型的定义就知道怎么用

安装

全局安装

npm install -g typescrip

安装完成后就可以创建ts文件了，然后利用tsc 文件名 来将其转换为js 文件，

tsc 文件名

创建配置文件 ：在项目根目录中控制台中，利用 tsc –init 创建tsconfig.json 文件

 tsc --init

启动监听

tsc -w

点击菜单 任务–运行任务 点击tsc 监视tsconfig.json 然后就可以自己生成代码了

数据类型

为了使编写代码更规范，更利用维护，增加了类型校验 写ts代码必须指定类型

布尔类型（ boolean ）

var flag:boolean = true;

数字类型 （number）

var num:number = 23;

字符串类型 （string）

var str:string = "DanielLee"

数组类型 ( array )

定义数组两种定义方式：

 1.  var arr: number[] = [11,22,33]   表示所有数组的元素都必须是数字
 2.  var arr: Array<number> = [11,22,33]  泛型   所有数组的元素都必须是数字

元组类型 （tuple） 属于数组的一种

let arr: [number,string] = [123,'this is ts']  给数组种每一个位置指定类型

枚举类型 （enum）

enum Flag {
    success =1,
    error = 2
};
let s:Flag = Flag.success;
console.log(s) // 1

enum color {blue,red,'orange'}
let c:color = color.red
console.log(c)  //1 如果标识符没有赋值，他的值就是下标

var c:color = color.orange;
console.log(c)  //4

只要改变了一个元素的值，其后面元素的值就会接着前面的值加1，只影响改变后面的元素

任意类型 （ any ）

var num: any = 124;
console.log(num)  //124
num = 'str';
console.log(num) //str

void 类型: 表示没有任何类型。一般用于定义方法的时候方法没有返回值

function run():void{
    console.log('rum')
}

如果方法有返回值，则方法返回什么，类型就设置为啥
function run():number{
    return 123
}
run();

never 类型 (其他类型)的子类型，代表从不会出现的值 包括null和undefined

这意味着声明never的变量只能被never 类型所赋值

var a:undefined   a = undefined
var b:null;  b = null;

var a:never 
a=(()=>{
    throw new Error('错误')
})()

null 和 undefined （其他类型的子类型）

var num:undefined;
console.log(num) // undefined

var num:number | undefinrd;     
num=123;
console.log(num) // 123   num 为undefined 或者num 都不会报错，这属于联合类型 用 | 

var num:number | undefinrd;
console.log(num)  // undefined
定义未赋值，为undefined

联合类型

var num:null;
num=null;

若一个元素可能是number类型 或者是null 或者为undefined
var num: number | null | undefined;
num = 1234;  console.log(num)

函数定义(定义函数)

函数声明法

 function run():string{
     return 'run'
 }

表达式定义法

var fun2 = function():number{
    return 123
}

ts中定义方法传参

function getInfo(name:string,age:number):string{
    return `${name} --- ${age}`
}
console.log(getInfo('zhangsan',20))

vara getInfo = function(name:string,age:number):string{
    return `${name} --- ${age}`
}
console.log(getInfo('zhangsan',40))

方法的可选参数（ts中的实参和形参必须一至，如果不一样（可传可不传）就需要配置可选参数）在参数后面加一个 ？ 注意：可选参数必须配置到参数的最后面

function getInfo(name:string,age?:number):string{
    if(age){
        return `${name} --- ${age}`
    }
    return `${name} --- 保密`
}
getInfo('zhangsan')

默认参数

function getInfo(name:string,age:number=20):string{
    if(age){
        return `${name} --- ${age}`
    }
    return `${name} --- 保密`
}
getInfo('zhangsan')  //zhangsan---20
getInfo('zhangsan',30)  //zhangsan --- 30

剩余参数

...运算符  接受实参传过来的值

function sum(...result:number[]):number{
    var sum = 0;
    for(var i = 0;i<result.length;i++){
        sum+=result[i];
    }
    return sum
}
alert(sum(1,2,3,4,5,6))

函数的重载 （同名的函数传入不同的参数实现不同的结果）

function good(name:string):string;
function good(age:number):number;
function good(str:any):any{
  if(typeof str === 'string'){
    return '我的名字'+ str
  }else{
    return '我的年龄是' + str
  }
}

console.log(good('lalala'))  //正确
console.log(good(123))     //正确
console.log(good(true))   //错误写法


function good(name:string):string;
function good(name:string,age:number):number;
function good(name:any,age?:any):any{
  if(age){
    return '我的名字'+ name + '我的年龄' + age 
  }else{
    return '我的名字' + name
  }
}
good('zhangsan')

typescript 定义类

    class Person{
       public name:string;   //属性  前面省略了public 关键词
        constructor(n:string){ //构造函数
            this.name = n
        }
        run():void{
            alert(this.name)
        }
        getName():string{
            return this.name;
        }
        setName(name:string):void{
            this.name = name;
        }
        }
 var p = new Person('张三')
 p.run()
